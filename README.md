# 🎬 Ultimate Video Edit Tools

A lightweight collection of Windows batch tools for video editing automation.  
Easily create shorts from images, merge videos, convert shorts into long videos, and cut long sequences into smaller clips.  

---

## ✨ Features
- 🖼 Create short videos from images  
- 🎥 Convert multiple shorts into long videos  
- 🔗 Join multiple video files seamlessly  
- ✂️ Cut long videos into smaller sequences  

---

## 🚀 How to Use
1. Install [FFmpeg](https://ffmpeg.org/) and add it to your system PATH.  
2. Clone this repo:  
   ```bash
   git clone https://github.com/username/video-tools-suite.git
   ```  
3. Run the batch scripts directly from Command Prompt or double-click.  

---

## 📂 Included Files
- `cut_videos` → Put your large video here from which to cut smaller video clips. [mp4, mkv, avi, mov, webm]  
- `images` → Put all your photos here from which you want to create all the shorts videos. [jpg, jpeg, png] 
- `join_videos` → Here you can put all your short videos that you want to combine with each other. [mp4, mkv, avi, mov, webm] 
- `output` → You will find the output videos of all the tools in this folder. 
- `temp` → Your projects will be saved in this file while running, and the temporary files will be automatically deleted from this folder when editing is finished. 
- `root` → Here you can put your Music [audio.mp3], background music [bg.mp3] and video [video.mp4], but make sure to keep the file names similar, otherwise the coding will not work. 

---

## 📂 Included Tools
- `create_short_from_images.bat` → Make short video from images  
- `shorts_to_long.bat` → Convert shorts into long videos  
- `join_videos.bat` → Merge multiple videos  
- `cut_long_video.bat` → Split long videos into smaller clips
